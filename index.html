Welcome

MeetMeCafe webapp helps managing meeting at your local area. You can see public meetings created for everybody, private meetings where you are invited, as well as you can create your own meetings. Try it. It's easy!

How to use it

Sign in to start seeing and managing meatings nearby. If you don't have account yet - sign up, it's easy.

After signing in you will see the map with meeting points around.

If you don't see your location - try to autolocate by clicking the button at the top right corner of the map, or just drag map to your location.

You can see the whole list of today meeting at the "My meetings page". There are private meetings (where you are the author or invited person), and public meetings - for everybody.

You can create your own meeting at the "New meeting" page. Just simply provide all the required data, choose the location, select who will attend the meeting and that's all.

If you create public meeting - everyone who signs in will see it. If you create private meeting - only the friends (invited by email) will see them. Even if your friends don't sign up yet - they will see all meeting where they are invited after signing up.

About the project

MeetMeCafe web application is created as a project for Udacity Senior Web Developer Nanodegree programm.

Currently it's working project, but of course there are a lot of things that need to be improved (and I hope to do that soon).

I'm using here HTML5, CSS3, JavaScript + Jquery, Bootstrap, FiraBase and Google Maps API. Icons for the map markers are created using Maps Icons Collection.

This web application is created completely by Ruslan Dorozhovets (rdesigned), using some of examples from API providers as well as idea solutions from StackOverflow.
